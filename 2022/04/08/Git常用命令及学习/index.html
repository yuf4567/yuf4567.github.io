Git常用命令及学习

发表于2022-04-08|更新于2022-09-03|前端学习
|阅读量:

今天在做上尚硅谷的vue电商项目，想着想把它放在github上面维护，以后在面试的时候拿得出手

所以总结学习了一些git的常用命令

1、git有三个工作区

编辑

  • 工作区：就是你在电脑里能看到的目录。
  • 暂存区：英文叫 stage 或 index。一般存放在 .git 目录下的 index 文件（.git/index）中，所以我们把暂存区有时也叫作索引（index）。
  • 版本库：工作区有一个隐藏目录 .git，这个不算工作区，而是 Git 的版本库。

2、git的一些常用命令

  • git init:初始化一个git仓库
  • git clone:从现有git仓库中拷贝项目到本地
  • git config:显示当前仓库配置信息
  • git config -e:修改当前仓库配置信息

编辑

  • git add

  • git status

  • git diff

  • git commit

  • git reset

  • git rm

    远程操作：

  • git pull

  • git push

  • git fetch
